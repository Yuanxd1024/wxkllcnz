### 注册机使用指南

1. **确保接码平台余额充足！！**
   最好是注册预算余额的两倍以上。

2. **触发注册流程**
   点击网页上方 `Actions`，选择“嘉立创注册机”，点击 `run workflow` 填写注册数量、密码等信息。
   
   > **关于“邮箱递增”的说明：**
   > 如果设置了递增数为 `5`，则实际绑定邮箱为 `xxxx5@2925.com`，并依次递增。
   > 
   > ⚠️ **注意：**每次运行前，都要看一下上一次绑定到哪个邮箱了。比如说上一次运行，最后一个账号绑定到了 `xxxx50@2925.com`，那么这次的递增数就要填 `51`。

3. **一次只运行一个任务**
   运行多个任务会导致冲突。 
   *提示：点击 `run workflow` 后，有时看似没反应，实际上已经触发成功，刷新网页看一下即可。*

4. **等待推送结果**
   坐等企业微信推送注册成果（如果 24 小时以内没有推送，最好回来查看一下运行日志）。


5. **及时把新账号整理到云笔记**
   我看到之后会把它们加到自动签到
